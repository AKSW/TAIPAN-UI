Statistical tables have two (or more) dimensions with numerical data inside cells. Statistical data has to be represented as cubes using for instance RDF DataCube vocabulary.

Not English

We do not consider any other languages, therefore if table is not in English mark it as "Not English".

Examples of subject columns

The subject column contains the entity, which connect to any other column via property. You should be able to represent any row as a set of triples with entity as a subject. Therefore, subject column is a center of a star-like graph.

The table below is talking about persons providing services. The subject column is column 0 (Name).
