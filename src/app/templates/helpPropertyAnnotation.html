Interface

Subject column of a table (i.e. column which contains entities) is marked with blue color. Your task is to define relationships between blue column and all others if they exist. To accomplish the task, you click on a column, type property in the input box (start typing "db..." to see suggestions from LOV API) and click add button. After you are done with a table click "Next Table" button, it will save your results and redirect you to the next table.
